# Semantic-Style-Pooler

This repository provides the experimental code for the following paper:

**From Semantics to Style: A Cross-Dataset Comparative Framework for Sentence Similarity Predictions** [https://aclanthology.org/2026.findings-eacl.95/](https://aclanthology.org/2026.findings-eacl.95/)

## Getting Started

The script `train_pooler.sh` serves as the primary entry point. It contains the configuration settings and initiates the training process for the pooler.
